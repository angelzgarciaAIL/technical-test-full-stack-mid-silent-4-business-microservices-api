# Instalación y Ejecución del Sistema de Microservicios

Este documento explica cómo instalar y ejecutar el sistema de microservicios desarrollado con Laravel y Node.js.

## Requisitos Previos

### Software Necesario:
- **PHP 8.1+** y Composer
- **Node.js 16+** y npm
- **MySQL 8.0+**
- **Git**

### Verificar instalaciones:
```bash
php --version
node --version
mysql --version
composer --version
